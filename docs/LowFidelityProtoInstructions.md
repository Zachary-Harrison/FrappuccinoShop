## Low Fidelity Prototype ##
In order to view the low fidelity prototype demonstrating the functionality of the page navigation system, do the following.

1. Clone the git hub repository.
2. Execute the following code in a command terminial from the repository: $python .\app\cs3450_group10\manage.py runserver 3000
3. Open a web browser and navigate to the following site: http://localhost:3000/shop_app/
